LynxHTTP - 一个模块化、高性能、轻量级的 C++ HTTP 服务器

LynxHTTP 是一个基于事件驱动的现代 HTTP 服务器，使用 C++ 开发，具备高并发、模块化与低资源占用等特性。

特性

  • 模块化架构：可插拔式路由、中间件和处理器
  • 高性能：非阻塞 I/O 与 Reactor 模式
  • 轻量级：无外部依赖的纯 C++ 实现
  • 线程池：扩展良好的任务处理模型

架构总览

  • 核心模块：TcpServer, TcpConn, EventLoop, HttpParser, ModuleManager
  • 并发模式：1 个 I/O 线程 + N 个工作线程
